## Template

When writing a handoff, include:

2-3 sentences describing what was accomplished.

### Verification

- UI to check
- Tests to execute

### Remaining
- Incomplete tasks
- Blockers encountered
- Known issues

### Next Session
- Setup steps needed
- Specific actions to take
- Context to reload
