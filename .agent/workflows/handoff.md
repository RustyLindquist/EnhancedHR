Workflow Name: Generate Task Summary for Handoff

Instructions:
- Review all progress to date, including key Artifacts (plans, Pre-Change Inventories, diffs, test results, screenshots, and recordings).
- Generate a concise Summary Artifact containing:
  - Completed tasks and outcomes.
  - Current project status and modified files.
  - Open items or next steps.
  - Key decisions, preserved functionality, and any resolved issues.
  - References to important existing Artifacts for continuity.
- Ensure the summary is self-contained, structured (e.g., sections for Status, Changes, Next Steps), and focused on facts to enable seamless onboarding of a new agent.
- Save relevant patterns or snippets to the knowledge base if applicable.
